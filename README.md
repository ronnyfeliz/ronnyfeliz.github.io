# Ronny Feliz — Portfolio

Personal portfolio CV — two versions:
- `index.html` → English version
- `index-es.html` → Spanish version

## Setup on GitHub Pages

1. Upload all files to your `ronnyfeliz.github.io` repository
2. Go to **Settings → Pages → Deploy from branch → main**
3. Your site will be live at `https://ronnyfeliz.github.io`

## File structure
```
/
├── index.html          (English CV)
├── index-es.html       (Spanish CV)
├── README.md
└── Pictures/
    └── Ronny Feliz Portrait 2.png  ← upload your photo here
```
